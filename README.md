# 🌱 Rootweave

A live, isometric regenerative-farming game. A once-thriving valley is **dying** — dust where wheat used to grow, dead trees, a muddy river. Bring it back to life: revive the dead ground, plant diverse crops, move animals through, shelter the river with buffer plantings, and watch the land, the water, and the wildlife return — field by field.

**▶ Play:** https://bryanralston.github.io/rootweave/

- Real regenerative principles as mechanics: diversity builds soil, sequenced grazing (cattle → chickens) creates synergy, cover crops and compost revive dead ground, and riparian buffers clean the river's runoff.
- Live real-time sim with pause + speed controls, seasons, data overlays (soil / water / biodiversity / runoff), region quests, and a shop.
- Touch-friendly — drag to pan, pinch to zoom. Works on phones and tablets.
- Self-contained: plain HTML5 + canvas, no dependencies. Just open `index.html`.

Built with Cortex.
